Ewidencja ludności
======

Projekt implementuje prostą usługę w architekturze klient-serwer, w której strona serwerowa przechowuje i udostępnia
interfejs do zdalnego zarządzania bazą danych ludności wraz z zależnościami małżeńskimi i rodzicielskimi pomiędzy
ludźmi.

####**Użyte technologie:**
    - Java RMI
